Python Version: 3.8 

Step 1: install modules/packages
    pip install -r requirements.txt
Step 2: run the script
    python audio-process.py <wav_file_path> <shot_file_path> <output_file_path>

    python audio-process.py meridian.wav MeridianShots.txt result.txt
